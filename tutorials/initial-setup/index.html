How to initially set up Alf.io

Ticket Categories

  • When you create an Event, you can set a sale period for each Ticket Category. This make sense especially for paid events like conferences, where you usually buy "Blind Bird", "Early Bird", "Full Price" tickets. You can define the sale period by defining the category Validity Range.
  • If you want to set a validity date for the tickets (and not for the category), you can do that by editing the "Advanced Options" for the category and set Custom ticket validity (will be displayed in ticket PDF, email and calendar invitation).

Stripe

  • The Stripe's secret key and Stripe's public key can be found on your Stripe Dashboard.
  • The Stripe Connect Client ID, Stripe Connect Callback URL & Stripe Connect WebHook signature key are only used when platform mode" has been activated, and the instance owner wants to collect fees from the organizers, and can just be left empty otherwise.
